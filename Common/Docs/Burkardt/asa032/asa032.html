ASA032
the Incomplete Gamma Function

ASA032 is which evaluates the incomplete Gamma function, by G Bhattacharjee.

ASA032 is Applied Statistics Algorithm 32. Source code for many Applied Statistics Algorithms is available through STATLIB.
