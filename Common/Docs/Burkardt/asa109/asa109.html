ASA109
Inverse of Incomplete Beta Function

ASA109 is which "inverts" the incomplete Beta function by Cran, Martin and Thomas.

Given a value of the incomplete Beta function, it can determine the argument that produced that value.

ASA109 is Applied Statistics Algorithm 109. Source code for many Applied Statistics Algorithms is available through STATLIB.
