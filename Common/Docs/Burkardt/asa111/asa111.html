ASA111
Percentage Points of the Normal Distribution

ASA111 is which computes the percentage points of the normal distribution, by Beasley and Springer.

ASA111 is Applied Statistics Algorithm 111. Source code for many Applied Statistics Algorithms is available through STATLIB.
