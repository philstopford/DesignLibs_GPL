ASA121
The Trigamma Function

ASA121 is which evaluates the trigamma function, by BE Schneider.

ASA121 is Applied Statistics Algorithm 121. Source code for many Applied Statistics Algorithms is available through STATLIB.

List of Routines:

  • TRIGAM calculates trigamma(x) = d**2 log(gamma(x)) / dx**2
  • TRIGAMMA_VALUES returns some values of the TriGamma function.
