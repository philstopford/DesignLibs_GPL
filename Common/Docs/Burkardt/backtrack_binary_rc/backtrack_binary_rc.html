BACKTRACK_BINARY_RC
Binary Backtrack Search Using Reverse Communication

BACKTRACK_BINARY_RC is which carries out a backtrack search for a set of binary decisions, using reverse communication (RC).

Related Data and Programs:

BISECTION_RC, which seeks a solution to the equation F(X)=0 using bisection within a user-supplied change of sign interval [A,B]. The procedure is written using reverse communication (RC).

CG_RC, which implements the conjugate gradient method for solving a positive definite sparse linear system A*x=b, using reverse communication (RC).

LOCAL_MIN_RC, which finds a local minimum of a scalar function of a scalar variable, without the use of derivative information, using reverse communication (RC), by Richard Brent.

ROOT_RC, which seeks a solution of a scalar nonlinear equation f(x) = 0, using reverse communication (RC), by Gaston Gonnet.

ROOTS_RC, which seeks a solution of a system of nonlinear equations f(x) = 0, using reverse communication (RC), by Gaston Gonnet.

SORT_RC, which can sort a list of any kind of objects, using reverse communication (RC).

SUBSET, which enumerates, generates, randomizes, ranks and unranks combinatorial objects including combinations, compositions, Gray codes, index sets, partitions, permutations, polynomials, subsets, and Young tables. Backtracking routines are included to solve some combinatorial problems.

ZERO_RC, which seeks solutions of a scalar nonlinear equation f(x) = 0, using reverse communication (RC).
