BLACK_SCHOLES
The Black-Scholes Equation

BLACK_SCHOLES is which demonstrates several simple approaches to the valuation of a European call using the Black-Scholes equation, by Desmond Higham.

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Related Data and Programs:

COLORED_NOISE, which generates samples of noise obeying a 1/f^alpha power law.

ORNSTEIN_UHLENBECK, which approximates solutions of the Ornstein-Uhlenbeck stochastic differential equation (SDE) using the Euler method and the Euler-Maruyama method.

PCE_ODE_HERMITE, a C++ program which sets up a simple scalar ODE for exponential decay with an uncertain decay rate, using a polynomial chaos expansion in terms of Hermite polynomials.

PINK_NOISE, which computes a "pink noise" signal obeying a 1/f power law.

SDE, which illustrates the properties of stochastic differential equations (SDE's), and common algorithms for their analysis, by Desmond Higham;

STOCHASTIC_DIFFUSION, which implements several versions of a stochastic diffusivity coefficient.

STOCHASTIC_RK, which applies a Runge-Kutta scheme to a stochastic differential equation.

List of Routines:

  • ASSET_PATH simulates the behavior of an asset price over time.
  • BINOMIAL uses the binomial method for a European call.
  • BSF evaluates the Black-Scholes formula for a European call.
  • FORWARD uses the forward difference method to value a European call option.
  • MC uses Monte Carlo valuation on a European call.
  • R8VEC_NORMAL_01_NEW returns a unit pseudonormal R8VEC.
  • R8VEC_PRINT_PART prints "part" of an R8VEC.
  • R8VEC_UNIFORM_01_NEW returns a unit pseudorandom R8VEC.
  • R8VEC_WRITE writes an R8VEC file.
