CC_TO_ST
Convert Sparse Matrix from Compressed Column to Sparse Triplet

CC_TO_ST is which converts sparse matrix information from compressed column (CC) to sparse triplet (ST) format.

Related Data and Programs:

CC, a data directory which contains examples of the Compressed Column (CC) sparse matrix file format;

CC_IO, which reads and writes sparse linear systems stored in the Compressed Column (CC) format.

ST, a data directory which contains examples of the Sparse Triplet (ST) format, a sparse matrix file format, storing just (I,J,A(I,J)), and using zero-based indexing.

ST_IO, which reads and writes sparse linear systems stored in the ST "sparse triplet" Sparse Matrix format.

ST_TO_CC, which converts a sparse matrix from sparse triplet (ST) to compressed column (CC) format.

Examples and Tests:

  • cc_to_st_test.txt, the output file.

List of Routines:

  • CC_PRINT prints a sparse matrix in CC format.
  • CC_TO_ST converts sparse matrix information from CC to ST format.
  • ST_PRINT prints a sparse matrix in ST format.
