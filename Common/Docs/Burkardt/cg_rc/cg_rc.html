CG_RC
Conjugate Gradient Method with Reverse Communication

CG_RC is which implements the conjugate gradient (CG) method for solving a positive definite sparse linear system A*x=b, using reverse communication (RC).

Related Data and Programs:

BACKTRACK_BINARY_RC, which carries out a backtrack search for a set of binary decisions, using reverse communication (RC).

BISECTION_RC, which seeks a solution to the equation F(X)=0 using bisection within a user-supplied change of sign interval [A,B]. The procedure is written using reverse communication (RC).

CG, which implements a simple version of the conjugate gradient (CG) method for solving a system of linear equations of the form A*x=b, suitable for situations in which the matrix A is positive definite (only real, positive eigenvalues) and symmetric.

JACOBI, which implements the Jacobi iteration for solving symmetric positive definite systems of linear equations.

LOCAL_MIN_RC, which finds a local minimum of a scalar function of a scalar variable, without the use of derivative information, using reverse communication (RC), by Richard Brent.

MGMRES, which applies the restarted GMRES algorithm to solve a sparse linear system.

ROOT_RC, which seeks a solution of a scalar nonlinear equation f(x) = 0, using reverse communication (RC), by Gaston Gonnet.

ROOTS_RC, which seeks a solution of a system of nonlinear equations f(x) = 0, using reverse communication (RC), by Gaston Gonnet.

SORT_RC, which can sort a list of any kind of objects, using reverse communication (RC).

SUPERLU, C++ programs which illustrate how a C++ program can call the SUPERLU library, (which is written in C), which applies a fast direct solution method to solve sparse linear systems, by James Demmel, John Gilbert, and Xiaoye Li.

TEST_MAT, which defines test matrices.

ZERO_RC, which seeks solutions of a scalar nonlinear equation f(x) = 0, using reverse communication (RC).
