FASTGL
Fast computation of Gauss-Legendre Quadrature Points and Weights

FASTGL is which carries out the fast computation of the K-th value and weight of an N-point Gauss-Legendre quadrature rule, by Ignace Bogaert.

Licensing:

The computer code and data files made available on this web page are distributed under the BSD license.

Related Data and Programs:

LEGENDRE_RULE, a C++ program which computes a Gauss-Legendre quadrature rule.

QUADMOM, which computes a Gaussian quadrature rule for a weight function rho(x) based on the Golub-Welsch procedure that only requires knowledge of the moments of rho(x).

QUADRULE, which contains information about quadrature rules, both as tabulated values, and as computational procedures.

TOMS655, which computes the weights for interpolatory quadrature rules; this library is commonly called IQPACK, by Sylvan Elhay and Jaroslav Kautsky.

Reference:

  1. Ignace Bogaert,
    Iteration-free computation of Gauss-Legendre quadrature nodes and weights,
    SIAM Journal on Scientific Computing,
