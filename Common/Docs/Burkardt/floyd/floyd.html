FLOYD
Shortest Distances Between Nodes in a Directed Graph

FLOYD is which demonstrates Floyd's algorithm for finding the shortest distance between every pair of nodes in a directed graph.

Related Data and Programs:

GRAFPACK, which computes various quantities associated with mathematical graphs.

GRAPH_REPRESENTATION, a data directory which contains examples of ways of representing abstract mathematical graphs

SUBSET, which generates, ranks and unranks various combinatorial objects.

TOMS097, which computes the distance between all pairs of nodes in a directed graph with weighted edges, using Floyd's algorithm.
