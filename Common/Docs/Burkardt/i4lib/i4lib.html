I4LIB
A Single Precision Integer Arithmetic Utility Library

I4LIB is which includes a number of utility routines for "I4" or single precision integer arithmetic.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Related Programs:

C4LIB, which implements certain elementary functions for "C4" or single precision complex variables;

C8LIB, which implements certain elementary functions for "C8" or double precision complex variables;

I8LIB, which contains many utility routines, using "I8" or "double precision integer" arithmetic.

R4LIB, which contains many utility routines, using "R4" or "single precision real" arithmetic.

R8LIB, which contains many utility routines, using "R8" or "double precision real" arithmetic.

SUBPAK, which contains many utility routines;
