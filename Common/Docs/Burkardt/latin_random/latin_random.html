LATIN_RANDOM
Latin Random Squares in M dimensions

LATIN_RANDOM is which makes Latin Random Squares.

A Latin square is a selection of one point from each row and column of a square matrix or table. In M dimensions, the corresponding item is a set of N points, where, in each dimension, there is exactly one point whose coordinates are in a given "column" or range of values. To emphasize the use of higher dimensions, these objects are sometimes called Latin hypersquares.

A Latin Random Square (I just made up this name) is a set of N points, where one point is taken at random from each of the subsquares of a Latin Square. These points may be regarded as an M dimensional quasirandom pointset.

Related Data and Programs:

BOX_BEHNKEN, which computes a Box-Behnken design, that is, a set of arguments to sample the behavior of a function of multiple parameters;

CVT, which computes points in a Centroidal Voronoi Tessellation.

FAURE, which computes Faure sequences.

GRID, which computes points on a grid.

HALTON, which computes elements of a Halton Quasi Monte Carlo (QMC) sequence, using a simple interface.

HAMMERSLEY, which computes elements of a Hammersley Quasi Monte Carlo (QMC) sequence, using a simple interface.

HEX_GRID, which computes sets of points in a 2D hexagonal grid.

IHS, which computes improved Latin Hypercube datasets.

LATIN_CENTER, which computes Latin square data choosing the center value.

LATIN_COVER, which produces N Latin squares which cover an NxN square, or NxN Latin cubes which cover an NxNxN cube.

LATIN_EDGE, which computes Latin square data choosing the edge value.

LATIN_RANDOM_DATASET, a C++ program which creates a Latin Random Hypercube dataset;

NIEDERREITER2, which computes Niederreiter sequences with base 2.

NORMAL, which computes elements of a sequence of pseudorandom normally distributed values.

SOBOL, which computes Sobol sequences.

UNIFORM, which computes uniform random values.

VAN_DER_CORPUT, which computes van der Corput sequences.
