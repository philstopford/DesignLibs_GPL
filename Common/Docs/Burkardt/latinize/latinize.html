LATINIZE
Force a Dataset to be a Latin Hypercube

LATINIZE is which adjusts an M dimensional dataset of N points so that it forms a Latin hypercube.

On output, each row of the table will have the properties that:

  1. the minimum and maximum row values are the same as on input;
  2. the row contains N evenly spaced values between the minimum and maximum.
  3. in each row, the elements retain their ordering.

  • cvt_02_00010.txt, the original set of 10 points in 2D.
  • cvt_02_00010.png, a PNG image of the points.
  • cvt_02_00010.latin.txt, the latinized version of the dataset.
  • cvt_02_00010.latin.png, a PNG image of the latinized points.
