LEVENSHTEIN
The Levenshtein Distance Between Two Strings

LEVENSHTEIN is which computes the Levenshtein distance between two strings.

Related Data and Programs:

CAESAR, which can apply a Caesar Shift Cipher to a string of text.

CHRPAK, which works with characters and strings.

ROT13, which can encipher a string using the ROT13 cipher for letters, and the ROT5 cipher for digits.

Source Code:

  • levenshtein_test.txt, the output file.

List of Routines:

  • LEVENSHTEIN_DISTANCE computes the Levenshtein distance between strings.
  • LEVENSHTEIN_DISTANCE_TEST tests LEVENSHTEIN_DISTANCE.
  • LEVENSHTEIN_MATRIX computes the Levenshtein distance matrix between strings.
  • LEVENSHTEIN_MATRIX_TEST tests LEVENSHTEIN_MATRIX.
