LINE_FELIPPA_RULE
Quadrature Rules for a Line Segment in 1D

LINE_FELIPPA_RULE is which generates the points and weights of a quadrature rule over the interior of a line segment in 1D.

Related Data and Programs:

CCN_RULE, a C++ program which defines a nested Clenshaw Curtis quadrature rule.

CHEBYSHEV1_RULE, a C++ program which computes and prints a Gauss-Chebyshev type 1 quadrature rule.

CHEBYSHEV2_RULE, a C++ program which computes and prints a Gauss-Chebyshev type 2 quadrature rule.

CLENSHAW_CURTIS_RULE, a C++ program which defines a Clenshaw Curtis quadrature rule.

LEGENDRE_RULE, a C++ program which computes a 1D Gauss-Legendre quadrature rule.

LINE_CVT_LLOYD, which applies Lloyd's iteration repeatedly to a set of N points, to compute a Centroidal Voronoi Tessellation (CVT) over the interior of a line segment in 1D.

LINE_FEKETE_RULE, which estimates the location of N Fekete points, for polynomial interpolation or quadrature, over the interior of a line segment in 1D.

LINE_GRID, which computes a grid of points over the interior of a line segment in 1D.

LINE_INTEGRALS, which returns the exact value of the integral of any monomial over the length of the unit line in 1D.

LINE_MONTE_CARLO, which applies a Monte Carlo method to estimate the integral of a function over the length of the unit line in 1D;

LINE_NCC_RULE, which defines a Newton Cotes Closed (NCC) rule using equally spaced points over the interior of a line segment in 1D.

LINE_NCO_RULE, which defines a Newton Cotes Open (NCO) rule using equally spaced points over the interior of a line segment in 1D.

PATTERSON_RULE, a C++ program which returns the points and weights of a 1D Gauss-Patterson quadrature rule of order 1, 3, 7, 15, 31, 63, 127, 255 or 511.
