LLSQ
Linear Least Squares Problem for Y = A*X+B

LLSQ is which solves the simple linear least squares (LLS) problem of finding the formula of a straight line y=a*x+b which minimizes the root-mean-square error to a set of N data points.

TEST_LLS, which implements linear least squares test problems of the form A*x=b.
