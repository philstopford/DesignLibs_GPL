PYRAMID_INTEGRALS
Integrals Inside the Unit Pyramid in 3D

PYRAMID_INTEGRALS is which returns the exact value of the integral of any monomial x^i y^j z^k over the interior of the unit pyramid in 3D.

The unit pyramid has a square base of area 4, and a height of 1. Specifically, the integration region is: 

        - ( 1 - Z ) <= X <= 1 - Z
        - ( 1 - Z ) <= Y <= 1 - Z
                  0 <= Z <= 1.
The volume of the unit pyramid is 4/3.
