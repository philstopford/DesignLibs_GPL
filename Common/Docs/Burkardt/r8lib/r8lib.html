R8LIB
A Double Precision Real Arithmetic Utility Library

R8LIB is which contains a number of utility routines for "R8" or "double precision real" arithmetic.

Licensing:

The computer code and data files made available on this web page are distributed under the GNU LGPL license.

Related Programs:

C4LIB, which implements certain elementary functions for "C4" or single precision complex variables;

C8LIB, which implements certain elementary functions for "C8" or double precision complex variables;

I4LIB, which contains many utility routines, using "I4" or "single precision integer" arithmetic.

I8LIB, which contains many utility routines, using "I8" or "double precision integer" arithmetic.

L4LIB, which contains many utility routines, using one byte logical (L4) variables.

R4LIB, which contains many utility routines, using "R4" or "single precision real" arithmetic.

R8COL, which contains utility routines for R8COL's, that is, double precision real MxN arrays, considered as N column vectors, each of length M. The data may be thought of as a matrix of multiple columns, and many operations will be carried out columnwise.

R8ROW, which contains utility routines for R8ROW's, that is, double precision real MxN arrays, considered as M row vectors, each of length N. The data may be thought of as a matrix of multiple rows, and many operations will be carried out rowwise.
