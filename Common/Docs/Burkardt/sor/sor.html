SOR
Linear System Solution by Successive Over-Relaxation

SOR is which implements a simple version of the successive over-relaxation method for the iterative solution of a system of linear equations.

Related Data and Programs:

JACOBI, which implements the Jacobi iteration for solving symmetric positive definite systems of linear equations.

TEST_MAT, which defines test matrices.

List of Routines:

  • DIF2 returns the DIF2 matrix.
  • R8MAT_MV multiplies a matrix times a vector.
  • R8MAT_RESIDUAL_NORM returns the norm of A*x-b.
  • R8VEC_COPY copies an R8VEC.
  • R8VEC_DIFF_NORM_SQUARED: square of the L2 norm of the difference of R8VEC's.
  • R8VEC_PRINT prints an R8VEC.
