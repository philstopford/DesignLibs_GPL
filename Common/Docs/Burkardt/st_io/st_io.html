ST_IO
Read and Write Sparse Triplet (ST) Matrix Files.

ST_IO is which reads and writes files in the Sparse Triplet (ST) format used for storing sparse matrices.

Related Data and Programs:

CC_IO, which reads and writes sparse linear systems stored in the Compressed Column (CC) format.

CC_TO_ST, which converts a sparse matrix from compressed column (CC) to sparse triple (ST) format.

HB_IO, which reads and writes sparse linear systems stored in the Harwell Boeing (HB) format for sparse matrices.

ST, the format which is used for the input to this program.

ST_TO_CC, which converts sparse matrix data from Sparse Triplet (ST) format to Compressed Column (CC) format.

Examples and Tests:

  • st_io_test.txt, the output file.
  • a5by5_r8.st, an ST file storing a 5 by 5 matrix.
  • kershaw_r8.st, an ST file storing a 4 by 4 matrix.
