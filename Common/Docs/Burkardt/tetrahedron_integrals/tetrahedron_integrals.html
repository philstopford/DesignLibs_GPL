TETRAHEDRON_INTEGRALS
Integrals Inside the Unit Tetrahedron in 3D

TETRAHEDRON_INTEGRALS is which returns the exact value of the integral of any monomial over the interior of the unit tetrahedron in 3D.

The interior of the unit tetrahedron in 3D is defined by 

        0 <= x
        0 <= y
        0 <= z
             x + y + z <= 1

The integrands are all of the form 

        f(x,y,z) = x^e1 * y^e2 * z^e3
where the exponents are nonnegative integers.
