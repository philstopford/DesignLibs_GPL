TOMS112
Point in Polygon

TOMS112 is which determines whether a point is inside a polygon.

Related Software and Data:

GEOMETRY, a C++7 library which performs geometric calculations in 2, 3 and N dimensional space.

POLYGON_MONTE_CARLO, which applies a Monte Carlo method to estimate the integral of a function over the interior of a polygon in 2D.

POLYGON_PROPERTIES, which computes properties of an arbitrary polygon in the plane, defined by a sequence of vertices, including interior angles, area, centroid, containment of a point, convexity, diameter, distance to a point, inradius, lattice area, nearest point in set, outradius, uniform sampling.

POLYGON_TRIANGULATE, which triangulates a possibly nonconvex polygon, and which can use gnuplot to display the external edges and internal diagonals of the triangulation.
