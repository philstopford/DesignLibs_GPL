TOMS515
K subsets of an N set

TOMS515 is which lists the subsets of size K selected from a set of size N.

Related Data and Programs:

SUBSET, which enumerates combinations, partitions, subsets, index sets, and other combinatorial objects.

Reference:

  1. Bill Buckles, Matthew Lybanon,
    Algorithm 515: Generation of a Vector from the Lexicographical Index,
    ACM Transactions on Mathematical Software,
    Volume 3, Number 2, June 1977, pages 180-182.
