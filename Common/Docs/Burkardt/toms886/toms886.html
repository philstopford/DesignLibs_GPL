TOMS886
Interpolation over the Rectangle, Ellipse, or Triangle

TOMS886 is which implements an interpolation procedure based on "Padua points", defined in the square [-1,+1]^2, whose interpolating power is especially good. It is possible to map these points to the general rectangle, ellipse or triangle to do interpolation on these regions as well.

The original, true, correct version of ACM TOMS Algorithm 886 is available through ACM: http://www.acm.org/pubs/calgo or NETLIB: http://www.netlib.org/toms/index.html.

Related Data and Programs:

LAGRANGE_INTERP_2D, which defines and evaluates the Lagrange polynomial p(x,y) which interpolates a set of data depending on a 2D argument that was evaluated on a product grid, so that p(x(i),y(j)) = z(i,j).

PWL_INTERP_2D, which evaluates a piecewise linear interpolant to data defined on a regular 2D grid.

PWL_INTERP_2D_SCATTERED, which evaluates a piecewise linear interpolant to datwhich is available at an irregularly arranged set of points.

RBF_INTERP_2D, which defines and evaluates radial basis function (RBF) interpolants to 2D data.

SHEPARD_INTERP_2D, which defines and evaluates Shepard interpolants to scattered 2D data, based on inverse distance weighting.

TEST_INTERP_2D, which defines test problems for interpolation of regular or scattered data z(x,y), depending on a 2D argument.

TRIANGLE_INTERPOLATE, which shows how vertex data can be interpolated at any point in the interior of a triangle.

VANDERMONDE_INTERP_2D, which finds a polynomial interpolant to data z(x,y) of a 2D argument by setting up and solving a linear system for the polynomial coefficients, involving the Vandermonde matrix.

Examples and Tests:

  • ellipse_output.txt, the output file.

RECTANGLE applies the procedure to a rectangle.

  • rectangle_output.txt, the output file.

TRIANGLE applies the procedure to a triangle.

  • triangle_output.txt, the output file.
