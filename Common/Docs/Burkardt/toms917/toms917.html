TOMS917
Evaluate Wright's Omega Function

TOMS917 is which evaluates Wright's Omega function, by Piers Lawrence, Robert Corless, David Jeffrey.

The original, true, correct version of ACM TOMS Algorithm 917 is available through ACM: http://www.acm.org/pubs/calgo or NETLIB: http://www.netlib.org/toms/index.html.
