TRIANGLE_INTEGRALS
Integrals of Polynomials over Triangles

TRIANGLE_INTEGRALS is which returns the exact value of the integral of a polynomial over the interior of an arbitrary triangle in 2D.

Related Data and Programs:

BALL_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit ball in 3D.

CIRCLE_INTEGRALS, which returns the exact value of the integral of any monomial over the surface of the unit circle in 2D.

CUBE_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit cube in 3D.

DISK01_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit disk in 2D.

HYPERBALL_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit hyperball in M dimensions.

HYPERCUBE_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit hypercube in M dimensions.

HYPERSPHERE_INTEGRALS, which returns the exact value of the integral of any monomial over the surface of the unit hypersphere in M dimensions.

LINE_INTEGRALS, which returns the exact value of the integral of any monomial over the length of the unit line in 1D.

POLYGON_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of a polygon in 2D.

PYRAMID_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit pyramid in 3D.

SIMPLEX_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit simplex in M dimensions.

SPHERE_INTEGRALS, which returns the exact value of the integral of any monomial over the surface of the unit sphere in 3D.

SQUARE_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit square in 2D.

TETRAHEDRON_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit tetrahedron in 3D.

TRIANGLE_ANALYZE, a C++ program which reads a triangle defined in a file, and uses the triangle_properties() library to compute angles, area, centroid, circumcircle, edge lengths, incircle, orientation, orthocenter, and quality.

TRIANGLE_DUNAVANT_RULE, which sets up a Dunavant quadrature rule over the interior of a triangle in 2D.

TRIANGLE_FEKETE_RULE, which defines Fekete rules for quadrature or interpolation over the interior of a triangle in 2D.

TRIANGLE_FELIPPA_RULE, which returns Felippa's quadratures rules for approximating integrals over the interior of a triangle in 2D.

TRIANGLE_LYNESS_RULE, which returns Lyness-Jespersen quadrature rules over the interior of a triangle in 2D.

TRIANGLE_MONTE_CARLO, which uses the Monte Carlo method to estimate the integral of a function over the interior of the unit triangle in 2D.

TRIANGLE_NCC_RULE, which defines Newton-Cotes closed quadrature rules over the interior of a triangle in 2D.

TRIANGLE_NCO_RULE, which defines Newton-Cotes open quadrature rules on a triangle.

TRIANGLE_SYMQ_RULE, which returns efficient symmetric quadrature rules, with exactness up to total degree 50, over the interior of an arbitrary triangle in 2D, by Hong Xiao and Zydrunas Gimbutas.

TRIANGLE_WANDZURA_RULE, which sets up a quadrature rule of exactness 5, 10, 15, 20, 25 or 30 over the interior of a triangle in 2D.

TRIANGLE01_INTEGRALS, which determines the exact integral of a monomial over the unit triangle in 2D.

WEDGE_INTEGRALS, which returns the exact value of the integral of any monomial over the interior of the unit wedge in 3D.

  • I4_TO_PASCAL converts a linear index to Pascal triangle coordinates.

  • POLY_POWER computes a power of a polynomial.

  • R8MAT_PRINT_SOME prints some of an R8MAT.
  • RS_TO_XY_MAP returns the linear map from reference to physical triangle.

  • TRIANGLE_MONOMIAL_INTEGRAL integrates a monomial over an arbitrary triangle.
  • TRIANGLE_POLY_INTEGRAL: polynomial integral over a triangle.

  • XY_TO_RS_MAP returns the linear map from physical to reference triangle.
