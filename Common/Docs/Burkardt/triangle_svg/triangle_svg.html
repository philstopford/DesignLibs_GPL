TRIANGLE_SVG
Plot Triangle and Points with Scalable Vector Graphics (SVG)

TRIANGLE_SVG is which uses Scalable Vector Graphics (SVG) to create a plot of a triangle and any number of points. Such plots can illustrate quadrature rules or sampling methods for the triangle.

Related Data and Programs:

TRIANGULATION_SVG, a C++ program which creates a Scalable Vector Graphics (SVG) image of a triangulation, which can be displayed by a web browser.
